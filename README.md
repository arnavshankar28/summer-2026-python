# Summer 2026 Python Projects

This repository contains projects I completed while learning Python and preparing for a sports analytics research project.

## Projects

### Jersey Cost Calculator
Calculates the total cost of an sports jersey including New Jersey sales tax.

### NBA Ticket Recommendation Tool
Recommends seating sections based on a fan's budget.

### NBA Player Database from Kaggle
Has every NBA player along with statistics and rankings

### Player Efficiency
Calculates a player's efficiency based on points scored, rebounds, and assists

### 6 Graphs
Each graph shows a perticular NBA team statistics such as total wins, total losses, points per game, total points per team, total payroll per team, and the salaries of the top 5 NBA players based off MVP voting ranked.
